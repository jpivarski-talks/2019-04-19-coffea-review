\begin{frame}{The value of a real physics use-case can't be overstated}
\large
\vspace{0.5 cm}
I've been trying to get ``embedded'' with a physics group like this for several years. (For example, XENON-nT.)

\vspace{0.5 cm}
\uncover<2->{My columnar analysis work had been significantly less real before Coffea ramped up last fall. Ben's work began this year. Andrew's picked up from where it had left off on the CMS Big Data Project.}

\vspace{0.5 cm}
\uncover<3->{So far, in my work, Ben's, and Andrew's, Coffea has been \underline{\it the} real-world use-case. In IRIS-HEP Analysis Systems, we are expanding to include at least one ATLAS analysis, but so far nearly all of the visible momentum has been from this project.}
\end{frame}
